# 888 Causeway React Website

A modern React recreation of the [888 Causeway Co-operative](https://www.888causeway.coop/) website.

## About 888 Causeway

888 Co-operative Causeway is an affordable, member-run co-working network for small businesses and organisations in the social economy, located in Melbourne CBD.

## Design Highlights

- Elegant typography pairing (Cormorant Garamond + DM Sans)
- Warm coral/gold and sage green color palette
- Subtle noise texture overlay for depth
- Floating gradient orbs with gentle animations
- Smooth hover effects on cards and buttons
- Fixed navigation that changes on scroll
- Fully responsive layout

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- React 18
- Vite
- CSS-in-JS (embedded styles)
- Google Fonts
